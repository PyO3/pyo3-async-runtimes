Function pyo3_async_runtimes::tokio::re_exports::pending

pub fn pending<T>() -> Pending<T>
Expand description

re-export pending to be used in tokio macros without additional dependency Creates a future which never resolves, representing a computation that never finishes.

The returned future will forever return Poll::Pending.

§Examples

ⓘ
use futures::future;

let future = future::pending();
let () = future.await;
unreachable!();
